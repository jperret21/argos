Project Goals

From precision imaging to automated variable star photometry

Desktop-Grade Control

Replace the mobile app with a precision desktop interface. Full keyboard and mouse control, dockable panels, and real-time feedback for every action.

Science-Grade FITS

Produce fully-compliant FITS files with complete headers. Compatible with PixInsight, Siril, and AstroImageJ out of the box.

Open Standards

Built on the ASCOM Alpaca protocol and the FITS standard. No proprietary formats, no vendor lock-in. Your data belongs to you.

Desktop Native

Built with PyQt6 for macOS. Responsive, multi-threaded, and designed for long observation sessions. No browser, no latency.

Session Automation

Automate light, dark, flat, and bias sequences. Plan your session, start it, and let the software handle the rest.

Community-Driven

Open source from day one. Built by astrophotographers, for astrophotographers. Contributions, feedback, and ideas welcome.

Photometry & Variable Stars

Designed for differential photometry. Automate repeated observations of variable stars, exoplanet transits, and other time-domain targets over weeks and months.

Architecture

Three layers with strict separation of concerns

UI Layer
seercontrol/ui/

PyQt6 panels and widgets. Presentation only — no business logic, no network I/O.

Panels Widgets Theme Shell
Worker Layer
seercontrol/workers/

QThread workers bridging core and UI. Communication via Qt signals only.

Exposure Polling Discovery Sequence
Core Layer
seercontrol/core/

Pure Python business logic. No PyQt6 imports. Testable without a display.

Alpaca Imaging Config Native Client

Threading Model

Main Thread Qt UI event loop — never blocks
Qt Signals
QThreads Network, disk, computation
