Argos

Desktop photometry for the ZWO Seestar S30 Pro.

View on GitHub Get started

The Seestar S30 Pro saves JPEGs with no metadata.
Argos writes science-grade FITS, measures every star,
and builds differential light curves for the AAVSO.

Capabilities

Telescope control ASCOM Alpaca — GoTo, tracking, park
Live preview Continuous exposure loop
FITS export 16-bit, science headers
Star detection HFD, FWHM, eccentricity
Plate solving ASTAP — WCS on every frame
Aperture photometry Sky annulus, CCD equation
Differential photometry Ensemble (Honeycutt 1992)
AAVSO export Extended File Format
Autofocus V-curve routines
Filter wheel Alpaca integration

Try it

Works with the ASCOM simulator — no telescope needed.

terminal 
brew install uv
git clone https://github.com/jperret21/argos.git
cd argos
uv sync --extra dev

# Terminal 1 — mock server
uv run python scripts/mock_alpaca_server.py

# Terminal 2 — Argos
uv run python main.py
