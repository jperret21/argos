Capabilities

Telescope control GoTo, tracking, park
Live preview Continuous exposure loop
FITS export 16-bit, full science headers
Star detection & focus metrics HFD, FWHM, eccentricity
ASTAP plate solving WCS on every frame
Aperture photometry Sky annulus, CCD equation
Ensemble differential photometry Honeycutt 1992 method
AAVSO light-curve export Extended File Format
Autofocus V-curve routines
Filter wheel Alpaca integration

Try it in two commands

No telescope? Use the built-in ASCOM simulator.

terminal 
brew install uv
git clone https://github.com/jperret21/seerstar.git
cd seerstar
uv sync --extra dev

# Terminal 1 — start the mock server
uv run python scripts/mock_alpaca_server.py

# Terminal 2 — launch Argos
uv run python main.py
