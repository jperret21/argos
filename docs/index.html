Argos

the hundred-eyed watcher

Desktop astrophotography & differential photometry
for the ZWO Seestar S30 Pro.

View on GitHub How to start

The Seestar shows you the sky.
Argos measures it.

The S30 Pro is a formidable little sky-explorer, and its app is a pleasure to use — but it’s built for looking, not for measuring. The FITS it saves are missing the metadata that real science needs. Argos picks up from there and turns this remarkable telescope into a scientific instrument: complete science headers, every star measured, astrometry on each frame, and differential light curves — on your laptop, as the light lands.

Argos Panoptes, the giant of a hundred eyes, was set to watch and never sleep — a fitting name for software that keeps watch over the sky.

Specs

From raw photons to a calibrated light curve — the whole chain runs on your laptop, frame by frame.

01 Acquisition & control
Mount & camera

ASCOM Alpaca GoTo, tracking and park — with the native Seestar jog API for the moves the firmware won’t expose over Alpaca.

Live view

Continuous exposure loop with linear / log / asinh auto-stretch. Display only — the linear frame written to disk is never touched.

Sequencer

Multi-step Light / Dark / Flat / Bias plans, flattened into one ordered, repeatable shot list.

Stellarium link

Slew the mount straight from Stellarium over Telescope Protocol v1.0; live pointing is reported back at ~1 Hz.

Autofocus

HFD V-curve sweep across focuser positions with a parabola-fit minimum. Focuser control is wired; the routine is landing next.

02 Frames & data
Science-grade FITS

Fills in the headers the Seestar leaves out — exposure mid-times, gain, airmass, Moon separation — so a frame is ready for Siril, PixInsight or AstroImageJ.

Green-plane measurement

Photometry and astrometry work on the raw green pixels directly — no demosaic, no interpolation, so every measurement is signal the camera actually recorded.

03 Astrometry
Star detection & QA

Sources at 5σ over a robust sky estimate, with HFD, Gaussian FWHM and eccentricity for up to 120 stars per frame.

Plate solving

ASTAP solves every sub against the sky and writes back a full WCS — so any pixel maps to real sky coordinates, even on short live frames.

04 Photometry
Aperture photometry

Circular aperture + median sky annulus → background-subtracted flux, instrumental magnitude and a CCD-equation error in electrons, with a saturation flag.

Ensemble differential

Zero-point from a comparison ensemble (mean of catalog − instrumental); the error folds the target’s photon noise into the ensemble scatter (Honeycutt 1992).

AAVSO catalog

VSX variables and VSP comparison stars by cone search on the solved centre; comparisons auto-ranked by proximity and matched brightness.

Light-curve export

Per-target CSV: JD_UTC and BJD_TDB, magnitude and error, airmass (Kasten–Young), FWHM, sky and comparisons used.

How to start

No telescope needed — Argos runs against the ASCOM Alpaca simulator.

terminal 
brew install uv
git clone https://github.com/jperret21/argos.git
cd argos
uv sync --extra dev

# Terminal 1 — mock server
uv run python scripts/mock_alpaca_server.py

# Terminal 2 — Argos
uv run python main.py

How to contribute

Argos is open source and built in the open.

Issues, ideas, and pull requests are all welcome — especially from Seestar owners who want to push their data further. The codebase is uv-managed, fully typed, and covered by tests.

terminal 
# fork & clone, then:
uv sync --extra dev
uv run --extra dev pytest
